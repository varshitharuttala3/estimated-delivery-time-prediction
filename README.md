# Estimated Delivery Time Prediction

## Problem Statement
Predict the estimated delivery time for orders using machine learning based on order and distance-related features.

## Dataset
The dataset includes order details, distance information, and time-related attributes.

## Approach
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature selection
- Model training and evaluation

## Models Used
- Linear Regression
- Random Forest Regressor

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab
- Gradio

## Results
The model predicts delivery time with reasonable accuracy.

## Future Improvements
- Add real-time traffic data
- Improve feature engineering
